Pastimes Clothing Store
------------------
Overview
-----------‐------
Pastimes Clothing Store is a web application developed in PHP and MySQL.
It allows users to browse second-hand clothing, manage a shopping cart, place orders, and view order history. Administrators can verify new users, edit or delete user accounts, and manage the platform.

---------
Features
----------

· User registration and login (email or username)
· Admin login with separate admin panel
· Browse products with images
· Add/remove items from shopping cart (session‑based)
· Checkout with mock payment (Cash on Delivery / simulated card)
· View order history with dates and totals
· Contact support form
· Admin dashboard: verify pending users, edit user details, delete regular users
· Local database storage using MySQL/MariaDB
· Input validation and error handling

-------------
Technologies Used
-------------

· PHP 7.4+
· MySQL (MariaDB)
· HTML5 / CSS3
· JavaScript (vanilla, for cart & admin interactions)
· Apache (XAMPP/WAMP)

-----------
How to Run
------------

1. Install XAMPP (or any local PHP/MySQL server)
2. Copy the pastimes folder into C:\xampp\htdocs\
3. Start Apache and MySQL from the XAMPP control panel
4. Open phpMyAdmin and create a database named ClothingStore
5. Visit http://localhost/pastimes/createTable.php to set up tables and sample data
6. Access the site:
   · User login: http://localhost/ASS/pastimes/login.php
   · Admin login: http://localhost/ASS/pastimes/admin_login.php
          (Default admin: admin@pastimes.com / admin123)
